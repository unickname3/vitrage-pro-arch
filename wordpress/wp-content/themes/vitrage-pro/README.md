# Vitrage Pro WordPress theme

## Быстрый старт
1. Скопировать тему в `wp-content/themes/vitrage-pro`.
2. Активировать тему в админке WordPress.
3. Скопировать каталог исходных статических ассетов в тему:
   - из репозитория: `assets/`
   - в тему: `wp-content/themes/vitrage-pro/assets/`
4. Создать меню и назначить локации:
   - `Primary menu`
   - `Footer menu`

## Что уже реализовано
- Регистрация CPT: `gallery_item`, `team_member`, `review_item`.
- Таксономия `gallery_category`.
- Базовые шаблоны темы для страниц/архивов/записей.
- Глобальные настройки контактов в разделе `Settings -> General`.

## Что настроить в админке
- Постоянные ссылки (`/%postname%/`).
- Главная страница (`front-page`).
- Контакты: телефон/email/адрес.
- Контент для CPT и страниц.
