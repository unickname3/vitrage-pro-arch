# Формы и UI/UX гайд для сайта-визитки

## Формы (админка без IT)
- Плагин: Contact Form 7 или Fluent Forms.
- Поля формы: Имя, Телефон (обяз.), Email, Сообщение.
- Настройки:
  - Получатель: бизнес-email заказчика.
  - Копия отправителю: включить.
  - Антиспам: reCAPTCHA v3/hCaptcha + honeypot.
  - Сообщение об успехе: явный текст + CTA на повторное действие.

## UX-правила
- Контраст текста не ниже WCAG AA.
- Все кликабельные элементы не ниже 44px по высоте.
- На мобильных: главный CTA в первом экране.
- Контакты всегда доступны в шапке/футере.
- Минимум обязательных полей в форме.

## Производительность
- Все изображения из медиатеки в современных форматах (WebP/AVIF).
- Lazy-load для карточек галереи.
- Уменьшение блокирующих скриптов, где возможно.

## Контрольные метрики
- Конверсия формы (submit rate).
- Доля мобильного трафика с отправкой заявки.
- Core Web Vitals по ключевым страницам.
