# Инструкция для заказчика (админка)

## Как менять контент

### 1. Контакты
- Перейти: `Settings -> General`.
- Заполнить поля:
  - Телефон
  - Email
  - Адрес
- Нажать `Save Changes`.

### 2. Галерея
- Перейти: `Галерея -> Добавить`.
- Указать заголовок, описание и изображение.
- Выбрать категорию.
- Нажать `Опубликовать`.

### 3. Команда
- Перейти: `Команда -> Добавить`.
- Указать ФИО, описание, фото.
- Нажать `Опубликовать`.

### 4. Отзывы
- Перейти: `Отзывы -> Добавить`.
- Указать заголовок и текст.
- Нажать `Опубликовать`.

### 5. Текстовые страницы
- Перейти: `Pages` и выбрать нужную страницу (например `Контакты`, `Цены`).
- Изменить текст.
- Нажать `Обновить`.

## Рекомендации по качеству
- Загружать изображения шириной 1600-2200px (без избыточного веса).
- Для каждого изображения заполнять alt-текст.
- Перед публикацией проверять страницу на мобильном.

## Регламент
- Делать backup перед массовыми правками.
- Обновлять плагины и WordPress не реже 1 раза в месяц.
