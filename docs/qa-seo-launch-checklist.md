# QA / SEO / Launch checklist

## 1) Функциональный QA
- [ ] Все страницы открываются без 404.
- [ ] Меню и внутренние ссылки корректны.
- [ ] Форма обратной связи отправляет письмо.
- [ ] Валидация полей формы работает на desktop/mobile.
- [ ] Контакты в шапке/футере актуальны.

## 2) Контент QA
- [ ] Перенесены тексты из исходных HTML.
- [ ] Перенесены изображения без битых ссылок.
- [ ] У карточек галереи и сотрудников есть превью.
- [ ] На страницах нет lorem/временных заглушек.

## 3) SEO checklist
- [ ] Настроены человеко-понятные URL.
- [ ] Настроены title/description на ключевых страницах.
- [ ] Включен `sitemap.xml` (SEO plugin).
- [ ] robots.txt корректный для прода.
- [ ] Настроены 301-редиректы со старых URL.

## 4) Performance
- [ ] Lighthouse mobile >= 70 (минимальная цель).
- [ ] Изображения оптимизированы.
- [ ] Кеш включен (плагин/хостинг).

## 5) Go-live
- [ ] Создан полный backup файлов и БД прода.
- [ ] Перенос темы и настроек на прод.
- [ ] Post-launch smoke test (главная, контакты, форма, 3-5 внутренних страниц).
- [ ] Мониторинг ошибок 24-48 часов после релиза.
